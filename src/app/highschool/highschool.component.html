Kart

End of studies project

My end of studies project was the construction of a kart directed by head movements.
Using these informations, the Arduino would send instructions to the motors moving the kart.
To test the code I started by logging values, but as it was not visual enough I linked the code to a Unity project and simulated the kart. Using this allowed me to have a visual of the sensors input and more accurately tune the code.
